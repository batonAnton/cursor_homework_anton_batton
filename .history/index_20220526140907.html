Basic
  • git config --global user.name blabla - налаштування імені користувача
  • git config --global user.email blabal@bla - налаштування мейлу користувача
  • git init - створює новий репозиторій Git
  • git status - показує стан файлів в каталозі,зміни які були залучені
  • git add - добавляє вміст каталога в індекс для подальшого коміта
  • git commit - добавляє коментар зі змінами
  • git log - перераховує всі коміти в репозиторію
  • git checkout - дозволяє переміщатись між гілками створеною командою git branch
  • git branch - дозволяє створювати,переглядати,перейменовувати і видаляти гілки
  • git push - дозволяє завантажувати вміст локального репозиторію у віддалений
  • git pull - дозволяє отримати вміст з віддаленого репозиторія до локального
  • git merge- дозволяє обєданти окремі гілки розробки в одну,створеною git branch
  • git clone - дозволяє зробити копію репозиторію

    Advanced

  • git revert - дозволяє безпечно відмінити зміни
  • git reset - дозволяє робити відміну локальних змін в репозиторію Git
  • git rebase- дозволяє переміщування,склеюванню і видалення комітів
  • git cherry-pick
  • git stash
